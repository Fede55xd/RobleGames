        <p><strong>RobleGames</strong> es una colección de juegos de navegador creados por RobleUY (@robleuy) que pueden ser jugados <strong>sin conexión a Internet</strong>. Están desarrollados utilizando <strong>HTML</strong>, <strong>CSS</strong> y <strong>JavaScript</strong>, lo que permite ejecutarlos directamente en tu navegador sin necesidad de instalación ni conexión activa.</p>

        <div class="section-title">🌟 Características</div>
        <ul>
            <li><strong>Juegos sin conexión:</strong> Todos los juegos de RobleGames se pueden jugar sin conexión a Internet. Solo necesitas un navegador web para disfrutarlos.</li>
            <li><strong>Ligero y rápido:</strong> Los juegos están diseñados para ser ligeros y rápidos.</li>
            <li><strong>Peso del instalador:</strong> Aproximadamente 10 MB</li>
        </ul>

        <div class="section-title">🚀 Juegos actuales</div>

                <em>Sobre el juego:</em> En <strong>DISPARALIENS</strong>, los jugadores deben elegir un mapa y una nave para enfrentarse a una invasión alienígena. El objetivo es eliminar a las naves enemigas mientras evitas que te dañen.
            </li>
            <li>
                <strong>ROBASTEROIDE</strong><br>
                <em>Género:</em> Acción / Espacial / Supervivencia<br>
                <em>Sobre el juego:</em> En <strong>ROBASTEROIDE</strong>, te adentrarás en el espacio exterior y deberás destruir cuantos asteroides puedas para conseguir el puntaje más alto posible.
            </li>
        </ul>
